Project Description

Multi Source Plagiarism Detection Techniques

Task invloves detecting plagiarism in source code when the code has been plagiarized from multiple sources. The current SOTA plagiarism detection tools take into consideration only a pair of programs and do not consider the aspect of multisource plagriarism which may prove to be a shortcoming.

Our approach involves extracting functions from programs using a parser and then passing them onto a transformer network to extract features. The next phase invloves extracting similarity scores and mapping function similarity scores with programs. A suitable algorithm is to be designed to get the overall probability.
